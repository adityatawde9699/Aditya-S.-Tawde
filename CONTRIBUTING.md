# Contributing to Aditya's Portfolio

Thank you for helping improve this project! Contributions can be made via issues, bug fixes, feature requests, documentation improvements, or tests.

## Getting started

1. Fork the repository.
2. Create a branch: `git checkout -b feature/your-feature`.
3. Set up the environment as described in README.
4. Run tests before committing:
   - Backend: `cd server && pytest`
   - Frontend: `cd client && npm test`

## Code style

- Python: follow [PEP 8], use `ruff` linting.
- JavaScript/React: use `eslint` + Prettier (via `npm run lint`).

## Submitting changes

- Open a pull request against `main`.
- Include screenshots or logs for UI and API changes.
- Add tests for your behavior.

## Branch naming

- `bugfix/<issue>`
- `feature/<short-description>`

## Communication

- Link to the issue in PR description.
- Keep updates clear and concise.
